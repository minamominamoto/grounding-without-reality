# Minimal Grounding Without Physical Reality

**Loop Closure and Resistance as Structural Conditions**

Minamo Minamoto · Independent Researcher  
ORCID: [0009-0002-1201-5704](https://orcid.org/0009-0002-1201-5704)

---

## Abstract

The symbol grounding problem, as formulated by Harnad (1990), ties
grounding to sensorimotor interaction with the physical world. This
paper argues that this formulation conflates two distinct claims: that
grounding requires a closed action–consequence loop with genuine error
possibility, and that such a loop must be anchored in physical reality.
We accept the first claim and contest the second.

On the view developed here, **minimal grounding** requires two
conditions: (i) a closed action–consequence loop, and (ii)
**resistance** — the possibility of action failure under constraints
independent of the agent's policy. Physical reality is one substrate
in which these conditions are realized, but not the only one.

We develop this through two cases: Rovee-Collier's mobile paradigm
(pre-locomotor infants discovering action–consequence regularities
before symbolic representation) and AlphaGo Zero (minimal grounding
instantiated in an abstract domain without physical reality). The
analysis exposes a **developmental inversion** in current multimodal
large language models, which acquire symbolic structure prior to any
grounded loop. We conclude with architectural implications linking
to the NAC, SAC, and abductive accumulation frameworks.

## Key Claims

1. **Substrate neutrality**: grounding does not require physical
   reality; it requires a loop with resistance.

2. **Minimal grounding**: the lower bound is any closed
   action–consequence loop in which the agent can fail under
   policy-independent constraints.

3. **Developmental inversion**: current multimodal LLMs acquire
   symbolic structure first and attach perception afterward —
   reversing the sequence in which loops precede symbols.

4. **Architectural implication**: a developmentally grounded
   architecture must begin with minimal loops and allow symbolic
   structure to emerge downstream of their retention, extension,
   and reorganization.
